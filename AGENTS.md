# AGENTS.md

This repository is designed for Codex autonomous development.

## Project goal
Create a polished browser-based FPS demo that runs locally and can deploy as a static site (GitHub Pages friendly).

## Engineering rules
- Prefer vanilla HTML, CSS, and JavaScript.
- Avoid heavy frameworks.
- Avoid build steps unless absolutely necessary.
- No backend services.
- Keep the project runnable by opening index.html locally.

## Scope discipline
Prioritise a complete playable experience over large feature sets.

If complexity grows, simplify features rather than leaving unfinished systems.

## Delivery workflow
- Create branch: feat/browser-fps-arena
- Implement the project
- Add README documentation
- Push branch
- Open pull request with summary and test steps
